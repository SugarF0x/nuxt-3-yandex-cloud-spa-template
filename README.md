# WORK IN PROGRESS

# Nuxt 3 SPA + Yandex Cloud Template

Nuxt 3 Yandex cloud deployment template with SPA mode & serverless functions 

## Development

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Deployment

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```
